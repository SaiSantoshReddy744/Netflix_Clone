# Netflix Clone

A beautiful, responsive Netflix clone with modern animations and effects.

## 🎯 Features

- **Responsive Design** - Works on all devices
- **Modern Animations** - Smooth hover effects and transitions
- **Netflix-like UI** - Authentic design and layout
- **Cool Effects** - Glowing cards, shimmer effects, 3D transforms
- **Interactive Elements** - Hover animations and visual feedback

## 🚀 Deployment

### GitHub Pages
1. Create a new repository on GitHub
2. Upload this folder to the repository
3. Go to Settings > Pages
4. Select source branch and save
5. Your site will be live at `https://username.github.io/repository-name`

### Netlify
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop this folder
3. Get instant deployment with custom domain

### Vercel
1. Connect your GitHub repository
2. Automatic deployments on every push
